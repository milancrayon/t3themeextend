# Insight – Blog System for TYPO3

Insight is a modern, flexible, and feature-rich blog extension for TYPO3 CMS.  
It helps you create and manage content-driven websites with ease, while also enabling AI-powered content creation directly inside TYPO3.

---

## 🚀 Features

### 📝 Blog Management
- Create, edit, and delete blog posts
- Category and subcategory support
- Tag management
- Author management

### 🌍 Content & Experience
- Multi-language support
- Author profile display
- Previous and next article navigation
- Social sharing integration

### 💬 Comment System
- Enable or disable comments
- Comment moderation and approval workflow

### ♿ Accessibility
- Text zoom in and zoom out
- Print-friendly layout

### 🤖 AI-Powered Content
- AI configuration support
- Assist content generation directly inside TYPO3 backend

### 🎨 Backend Flexibility
- Edit templates directly from TYPO3 backend
- No need to modify files manually
