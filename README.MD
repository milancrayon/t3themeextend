# T3theme – Modern Theme Engine for TYPO3

T3theme is a powerful, flexible, and feature-rich theme management extension for TYPO3 CMS.  
It provides a standardized framework for building custom content elements and managing site-wide theme configurations directly from the TYPO3 backend.

---

## 🚀 Features

### 🎨 Theme Configuration
- Centralized management for Logos, Favicons, and site settings
- Dynamic site branding via backend records
- Flexible page layouts (Default, Left Sidebar, Right Sidebar)

### 🧩 Custom Elements
- Streamlined creation of custom content elements
- Create TYPO3 Element Using Drag and Drop Easy User Interface.Provide field Name , Type and Custom Content Element Ready.

### 🧩 Container Elements
- Create Container Element for complex layout like grid
- Just add name & add column for create container element

### ⚙️ Developer Efficiency
- Fully compatible with **TYPO3 13 and 14**
- Native support for **Site Sets** configuration

---

## 📦 Installation

### Using Composer

```bash
composer require crayon/t3theme
```

---

## 🛠 Backend Flexibility
- Configure theme settings directly in the TYPO3 backend (via Theme Config records)
- Manage custom elements without modifying core template files
- Real-time previews and structured content editing
